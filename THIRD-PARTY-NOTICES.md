## JIVE

The `submodules/JIVE/` directory is a git submodule containing the JIVE
declarative UI framework, providing `juce::ValueTree`-based layouts, JSON style
sheets, and Flex/Grid adaptive layout for JUCE applications.

- **Repository**: <https://github.com/ImJimmi/JIVE>

## melatonin_inspector
