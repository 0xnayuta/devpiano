## 1. 测试目标

本测试文档主要验证以下内容：

- 电脑键盘到 MIDI note 的映射是否正确
- note on / note off 是否成对出现
- 长按与连按时是否稳定
- 焦点切换后按键状态是否残留
- 当前默认布局与布局标识恢复是否一致
- 最小布局操作入口（切换 / 保存当前布局 / 恢复默认布局）是否正常
- 从字符路径迁移到稳定 key code 主路径后，行为是否仍符合预期
